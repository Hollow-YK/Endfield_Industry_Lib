## 简介

一个 [Mindustry](https://github.com/Anuken/Mindustry) 模组，用于提供终末地的物品与设备。

> [!Caution]
>
> **模组正处于开发初期**，目前不建议用于游戏。

> [!Tip]
>
> 最低版本要求为 **144**

### 介绍

<details><summary>点击查看详细介绍</summary>

#### 添加设备

- 传送带、物品准入口、物流桥、汇流器、分流器

#### 添加物品

- 三种矿石(源矿、紫晶矿、蓝铁矿)
- 多种植物及种子

</details>
